History of the Domain Name System

Early Networking Before DNS

Before the modern internet existed, computer networks were small and mostly used by universities, government agencies, and research groups. As more computers were added, people needed a way to keep track of hostnames and IP addresses. At first, this was done manually using a single file called HOSTS.TXT, which had to be updated by hand.

The Creation of DNS

In 1983, Paul Mockapetris designed the Domain Name System (DNS). It replaced the old manual system with a hierarchy of servers that could answer name lookup requests automatically. DNS became an official internet standard in 1986.

Dns in the Modern Internet

Today, DNS is a core part of how the internet works. It has grown to support billions of devices, global traffic, and advanced features like security extensions and Anycast routing. Even though it has evolved, DNS still performs the same basic job, translating human readable names into IP addresses.
